## Next.js + RTK + tailwindCSS + nextAuth + prisma + mysql 로 만든 [KREAM](https://kream.co.kr/)사이트 만들기


## 시작하기

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

위 명령어를 실행 후 브라우저에 [http://localhost:3000](http://localhost:3000)로 접속 하면 위 코드 결과를 볼 수 있습니다.
